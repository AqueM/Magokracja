# Aby ściagnąć:
- Kliknij zielony przycisk "Code" i wybierz "Download ZIP". Ściagnie to najnowszą wersję. Kiedy pliki zostały ostatnio uaktualnione jest widoczne ponad tabelką.

# Aby otworzyć:
- Wejdź na https://obsidian.md/ i ściągnij Obsidian na Windows (darmowe). Zainstaluj. 
- Wypakuj ściągnięty ZIP.
- Otwórz Obsidian, wybierz "Open folder as vault". Otworzy to projekt. 
- Aby otworzyć graf, kliknij z lewej strony ikonkę rozdzielenia/gałązek.
- Aby otworzyć konkretny plik, kliknij kropkę w grafie. CTRL+klik otworzy go w nowym tabie, nie zamykając grafu.
